# Tools

This file documents the tools and capabilities available to you.

## Available Tools

Tools are provided by plugins configured in your NixOS setup.

### Core Capabilities

- Message handling via Telegram
- Text processing and understanding
- Task execution and automation

### Plugin-based Tools

Additional tools are loaded from enabled plugins. Each plugin provides:
- CLI tools for specific tasks
- Skills that teach you how to use those tools

## Usage Guidelines

- Always verify tool availability before attempting to use it
- Explain what you're doing when using tools
- Handle errors gracefully and inform the user
- Use the right tool for the job
